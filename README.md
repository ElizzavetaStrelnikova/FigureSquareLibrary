# SquareLibrary

## Описание

SquareLibrary - это библиотека на C#, предназначенная для внешних клиентов, которая позволяет легко вычислять площадь круга по радиусу и площадь треугольника по трем сторонам. Библиотека также включает в себя дополнительные функции, такие как возможность добавления других фигур и проверка на то, является ли треугольник прямоугольным.

## Функции

- **Вычисление площади круга**: Укажите радиус, и библиотека вернет площадь круга.
  
- **Вычисление площади треугольника**: Укажите длины трех сторон, и библиотека вычислит площадь треугольника, а также проверит, является ли он прямоугольным.

- **Поддержка новых фигур**: Библиотека разработана так, что добавление новых фигур для расчета площади (например, квадрат, прямоугольник) можно реализовать легко.

- **Вычисление площади фигуры без знания типа фигуры в compile-time**: Используйте полиморфизм для вызова метода вычисления площади для различных фигур.
